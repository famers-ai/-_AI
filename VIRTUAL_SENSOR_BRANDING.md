## 🚀 사용자 여정 (User Journey)

### Step 1: 첫 방문 (First Impression)
**목표**: "와, 센서 없이도 되네?"

**화면 구성**:
1. 대형 헤더: "Virtual Sensor System ✨ AI-Powered"
2. 정보 배너: "No sensors or hardware required!"
3. 가상 센서 카드: "Virtual Sensor Active" 배지

**심리적 효과**:
- 진입 장벽 제거 (비용 부담 없음)
- 기술적 신뢰성 (AI, Physics Model)
- 즉시 사용 가능 (Active 상태)

### Step 2: 데이터 확인 (Data Exploration)
**목표**: "정확한가?"

**화면 구성**:
1. VPD 값 표시: 큰 숫자로 명확하게
2. 상태 메시지: "✅ Optimal" 또는 "⚠️ Risk"
3. 설명 툴팁: "AI Physics Model" 상세 설명

**심리적 효과**:
- 데이터 신뢰성 확인
- 과학적 근거 이해
- 전문성 인식

### Step 3: 보정 참여 (Calibration)
**목표**: "내가 개선할 수 있네!"

**화면 구성**:
1. "AI Physics Model Running" 상태 표시
2. "Calibrate" 버튼 - 눈에 띄는 위치
3. "⚡ Calibrate for Better Accuracy" CTA

**심리적 효과**:
- 사용자 참여 유도
- 정확도 향상 가능성 제시
- 주인의식 형성

### Step 4: 신뢰 구축 (Trust Building)
**목표**: "계속 쓰고 싶다"

**화면 구성**:
1. AI 진단 패널: "Virtual Intelligence"
2. Confidence Score: "85% Confidence"
3. 실시간 업데이트: 애니메이션 효과

**심리적 효과**:
- 지속적인 가치 제공
- 투명한 신뢰도 표시
- 전문적인 분석 제공
